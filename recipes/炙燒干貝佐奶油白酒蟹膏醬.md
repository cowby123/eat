# 炙燒干貝佐奶油白酒蟹膏醬

生食級干貝表面炙燒，搭配奶香濃郁的蟹膏醬汁。無菜單開場第二道。

---

## 食材（約 20 份）

### 主體
| 材料 | 份量 | 備註 |
|------|------|------|
| 生食級干貝（不規則形） | 約 400–500g | 切厚片約 1.5–2cm |

### 奶油白酒蟹膏醬
| 材料 | 份量 |
|------|------|
| 無鹽奶油 | 40g |
| 蒜末 | 2 瓣 |
| 白酒 | 50ml |
| 鮮奶油 | 80ml |
| 蟹膏 | 2–3 大匙 |
| 鹽、白胡椒 | 適量 |

---

## 做法

### 奶油白酒蟹膏醬
1. 中小火融化奶油，加蒜末炒至微黃香
2. 倒入白酒，轉大火讓酒精揮發約 1 分鐘
3. 加入鮮奶油，轉小火收至略濃稠
4. 加入蟹膏攪拌均勻，以鹽、白胡椒調味
5. 醬汁保溫備用

### 炙燒干貝
1. 干貝切厚片，表面擦乾（重要：水分會影響炙燒效果）
2. 噴槍或超高溫鑄鐵鍋快速炙燒單面，至表面金黃焦香
3. 內部保持半生狀態

### 擺盤
1. 盤底淋少量蟹膏醬
2. 放上炙燒干貝（焦面朝上）
3. 再點綴少量醬汁於干貝表面
4. 可加少許蔥花或細香蔥增色

---

## 備注

- 醬汁可提前做好，上桌前加熱即可
- 干貝炙燒動作要快，避免過熟失去滑嫩感
- 蟹膏鹹度高，調味時先少放鹽，試味後再補
- 丁姊夫（甲殼類過敏）**不可食用此道**（含蟹膏）
